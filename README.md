> A long, long time from now, in the wake of what will no longer be called JavaScript, might be going to have existed a programming language called _kesh_.


```js
joe: {
    name: "Joe"
    age: 27
}

def greet(person):
    greeting: "Hey, {person.name}!"
    return greeting

greet(person: joe) // "Hey, Joe!"

```
